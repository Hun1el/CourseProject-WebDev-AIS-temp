# Автоматизированная информационная система по ведению базы данных разработки web-сайтов на базе платформы .NET.

**Платформа:** .NET Framework | **Язык:** C# | **БД:** MySQL | **Интерфейс:** Windows Forms

---

## 📋 Описание

Автоматизированная информационная система (АИС) предназначена для комплексной автоматизации деятельности организаций, занимающихся веб-разработкой. Система обеспечивает управление услугами, сотрудниками, клиентами, заказами и отчётностью по продажам.

**Основное назначение:**
- Оформление заказов на разработку веб-сайтов
- Ведение учёта выполненных услуг
- Формирование счётов для клиентов
- Подготовка отчётов для руководства

## 🎯 Основные возможности

### ✅ Система авторизации
- Защита доступа с помощью логина и пароля
- CAPTCHA-защита
- Поддержка трёх ролей: администратор, менеджер, директор

### 👥 Управление пользователями (Администратор)
- Добавление, редактирование, удаление пользователей
- Автоматическая генерация паролей
- Фильтрация и сортировка по ролям
- «Живой» поиск по фамилии
- Скрытие личных данных с возможностью раскрытия

### 📦 Управление услугами (Администратор)
- Добавление, редактирование, удаление услуг
- Добавление изображений к услугам
- Сортировка по цене
- Фильтрация по категориям
- «Живой» поиск по наименованию
- Расширенное описание услуг в модальном окне

### 🛒 Управление заказами (Менеджер)
- Оформление новых заказов происходит через корзину
- Добавление услуг в заказ
- Автоматический расчёт стоимости с учётом скидок и надбавок
- Редактирование статуса и сроков выполнения
- Формирование чеков в MS Word
- Просмотр состава заказа

### 👫 Управление клиентами (Менеджер)
- Добавление, редактирование, удаление клиентов
- Поиск и сортировка по фамилии
- Скрытие личных данных с возможностью раскрытия

### 📊 Отчётность (Директор)
- Просмотр всех заказов по периодам
- Формирование отчётов в MS Excel
- Фильтрация по статусу
- Сортировка по датам
- Поиск по номеру заказа

### 📚 Справочники (Администратор)
- Управление ролями
- Управление статусами заказов
- Управление категориями услуг
- Добавление, редактирование, удаление записей

## 🔒 Безопасность

- **Шифрование паролей:** SHA256
- **Разделение доступа:** 3 роли в системе
- **Защита данных:** Маскирование личных данных
- **Валидация входа:** CAPTCHA с блокировкой
- **SQL-защита:** Параметризованные запросы
- **Контроль операций:** Подтверждение удаления и редактирования

## 🚀 Быстрый старт

### Установка

1. **Скачайте установщик** и запустите его
2. **Выберите путь установки** (или используйте по умолчанию)
3. **Опционально:** создайте ярлык на рабочем столе
4. **Завершите установку** и запустите приложение

### Первый запуск

1. Откроется **форма авторизации**
2. Введите учётные данные администратора (по умолчанию: `admin` / `admin`)
3. При необходимости введите **CAPTCHA**
4. Вы получите доступ в зависимости от вашей роли

### Структура базы данных

Система использует 8 таблиц:

- **Orders** - основная таблица заказов
- **Users** - информация о пользователях
- **Clients** - данные клиентов
- **Product** - услуги/товары
- **OrderProduct** - состав заказов
- **Role** - роли пользователей
- **Category** - категории услуг
- **Status** - статусы заказов

## 👨‍💼 Функции по ролям

### 🔑 Администратор

**Доступные операции:**
- Управление пользователями
- Управление услугами с загрузкой фото
- Управление справочниками (роли, статусы, категории)
- Просмотр всех заказов
- Просмотр состава заказов

**Автоматизация:**
- Генерация паролей
- Капитализация имён
- Маска ввода для номеров телефонов

---

### 📱 Менеджер

**Доступные операции:**
- Управление клиентами
- Просмотр и фильтрация услуг
- Оформление заказов через корзину
- Редактирование заказов
- Печать чеков в Word
- Просмотр состава заказов

**Автоматизация:**
- Автоматическая дата заказа (текущая дата)
- Автоматическое назначение сотрудника
- Расчёт стоимости с скидками и наценками
- Сохранение черновиков в корзине

---

### 📈 Директор

**Доступные операции:**
- Просмотр всех заказов
- Формирование отчётов в Excel
- Фильтрация по периодам
- Сортировка и поиск заказов

**Аналитика:**
- Экспорт в Excel с применением фильтров
- Диапазон дат от первого до последнего заказа
